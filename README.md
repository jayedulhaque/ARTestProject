## About This App

- Download this App.
- create a mysql schema into your datbase by named "testing".
- configure the .env file
DB_DATABASE=testing
DB_USERNAME=your username
DB_PASSWORD=your password
- Run the command below:
php artisan migrate
- Run the artisan server in the project directory.
- Enjoy this app.

Laravel is accessible, powerful, and provides tools required for large, robust applications.
